# Smart Resume Shortlisting System

## Overview
An AI-powered recruitment platform that automates resume screening, candidate ranking, and job matching.

## Features
- Candidate Portal
- Recruiter Portal
- Admin Dashboard
- Resume Upload & Parsing
- OCR Support
- Resume Analysis Engine
- Candidate Matching & Ranking
- Gemini AI Evaluation
- Notification System

## Technology Stack
- ReactJS
- Tailwind CSS
- Python Flask
- MySQL
- JWT Authentication
- EasyOCR
- PyPDF2
- Gemini API

## Project Status
Currently under development. Core modules including authentication, frontend, backend APIs, database integration, and resume parsing have been completed. Resume analysis, matching engine, and AI evaluation modules are in progress.
